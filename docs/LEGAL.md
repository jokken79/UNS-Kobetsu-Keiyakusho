# 法的コンプライアンスガイド

## 労働者派遣法第26条 - 16項目要件

本システムは、労働者派遣法（労働者派遣事業の適正な運営の確保及び派遣労働者の保護等に関する法律）第26条に基づき、個別契約書に記載が必要な16項目を管理します。

---

## 必須16項目一覧

### 第1号〜第4号: 基本情報

| 項目 | 説明 | システム上のフィールド |
|------|------|------------------------|
| **1. 派遣労働者が従事する業務の内容** | 具体的な業務内容の記載 | `work_content` |
| **2. 業務に伴う責任の程度** | 補助的業務/通常業務/責任業務 | `responsibility_level` |
| **3. 派遣労働者が従事する事業所の名称・所在地** | 実際の就業場所 | `worksite_name`, `worksite_address` |
| **4. 派遣労働者を直接指揮命令する者** | 部署・役職・氏名 | `supervisor_department`, `supervisor_position`, `supervisor_name` |

### 第5号〜第8号: 就業条件

| 項目 | 説明 | システム上のフィールド |
|------|------|------------------------|
| **5. 労働者派遣の期間及び派遣就業をする日** | 派遣期間と就業日 | `dispatch_start_date`, `dispatch_end_date`, `work_days` |
| **6. 派遣就業の開始・終了時刻、休憩時間** | 就業時間の詳細 | `work_start_time`, `work_end_time`, `break_time_minutes` |
| **7. 安全及び衛生に関する事項** | 安全衛生措置 | `safety_measures` |
| **8. 派遣労働者からの苦情の処理** | 苦情処理担当者情報 | `haken_moto_complaint_contact`, `haken_saki_complaint_contact` |

### 第9号〜第12号: 契約管理

| 項目 | 説明 | システム上のフィールド |
|------|------|------------------------|
| **9. 派遣契約の解除に当たって講ずる措置** | 契約解除時の対応 | `termination_measures` |
| **10. 派遣元責任者・派遣先責任者** | 責任者情報 | `haken_moto_manager`, `haken_saki_manager` |
| **11. 派遣労働者を無期雇用・60歳以上に限定するか** | 派遣労働者の限定 | `is_mukeiko_60over_only` |
| **12. 派遣先が派遣労働者を雇用する場合の紛争防止措置** | 直接雇用防止措置 | `is_direct_hire_prevention` |

### 第13号〜第16号: 待遇・その他

| 項目 | 説明 | システム上のフィールド |
|------|------|------------------------|
| **13. 派遣労働者の数** | 派遣労働者の人数 | `number_of_workers` |
| **14. 労使協定方式の対象か否か** | 労使協定方式の適用 | `is_kyotei_taisho` |
| **15. 派遣先が福利厚生施設の利用機会を与える場合の措置** | 福利厚生施設 | `welfare_facilities` |
| **16. 派遣料金** | 時間単価等の料金情報 | `hourly_rate`, `overtime_rate`, `night_shift_rate`, `holiday_rate` |

## 法的根拠

### 労働者派遣法第26条（抜粋）

```
第二十六条　労働者派遣契約（当事者の一方が相手方に対し労働者派遣をすることを
約する契約をいう。以下同じ。）の当事者は、厚生労働省令で定めるところにより、
当該労働者派遣契約の締結に際し、次に掲げる事項を定めるとともに、その内容の
差異に応じて派遣労働者の人数を定めなければならない。

一　派遣労働者が従事する業務の内容
二　派遣労働者が労働者派遣に係る労働に従事する事業所の名称及び所在地
　　その他派遣就業の場所並びに組織単位
三　労働者派遣の役務の提供を受ける者のために、就業中の派遣労働者を
　　直接指揮命令する者に関する事項
四　労働者派遣の期間及び派遣就業をする日
五　派遣就業の開始及び終了の時刻並びに休憩時間
六　安全及び衛生に関する事項
七　派遣労働者から申出を受けた苦情の処理に関する事項
八　労働者派遣契約の解除に当たつて講ずる派遣労働者の雇用の安定を
　　図るために必要な措置に関する事項
九　労働者派遣契約が紹介予定派遣に係るものである場合にあつては、
　　当該職業紹介により従事すべき業務の内容及び労働条件その他の
　　当該紹介予定派遣に関する事項
十　前各号に掲げるもののほか、厚生労働省令で定める事項
```

## コンプライアンスチェックリスト

契約書作成時に以下を確認してください：

### 必須確認事項

- [ ] 業務内容が具体的に記載されているか
- [ ] 責任の程度が明確か
- [ ] 就業場所の名称・住所が正確か
- [ ] 指揮命令者が特定されているか
- [ ] 派遣期間が適切か（抵触日を超えていないか）
- [ ] 就業時間・休憩時間が明記されているか
- [ ] 時間外労働の上限が設定されているか
- [ ] 苦情処理担当者が両社で設定されているか
- [ ] 責任者が両社で設定されているか
- [ ] 派遣料金が適正に設定されているか

### 追加確認事項

- [ ] 安全衛生措置が記載されているか
- [ ] 契約解除時の措置が記載されているか
- [ ] 福利厚生施設の利用について記載されているか
- [ ] 労使協定方式の対象か否かが明記されているか

## 派遣期間の制限

### 事業所単位の期間制限
- 同一の事業所における派遣可能期間は原則3年
- 延長には過半数労働組合等への意見聴取が必要

### 個人単位の期間制限
- 同一の派遣労働者を同一の組織単位に派遣できる期間は3年

### 例外
以下の場合は期間制限の対象外：
1. 無期雇用の派遣労働者
2. 60歳以上の派遣労働者
3. 有期プロジェクト業務
4. 日数限定業務
5. 産前産後休業等の代替業務

## 罰則規定

### 法令違反時の罰則
- 許可取消し・事業停止命令
- 改善命令
- 公表制度
- 罰金（最高30万円〜100万円）

### 主な違反事例
1. 二重派遣
2. 期間制限違反
3. 禁止業務への派遣
4. 個別契約書の不備
5. 派遣元管理台帳・派遣先管理台帳の未整備
